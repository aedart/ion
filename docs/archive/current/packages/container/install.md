# How to install

## npm

```bash:no-line-numbers
npm install --save-peer @aedart/container
```

## yarn

```bash:no-line-numbers
yarn add --peer @aedart/container
```

## pnpm

```bash:no-line-numbers
pnpm add --save-peer @aedart/container
```
