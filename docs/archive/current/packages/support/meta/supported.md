# Supported Elements

The `meta` decorator supports the following elements¹:

* `class`
* `method`
* `getter`
* `setter`
* `field`
* `accessor`

¹: _An element is determined by the decorator's [`context.kind`](https://github.com/tc39/proposal-decorators#2-calling-decorators) property._
