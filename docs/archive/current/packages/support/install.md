# How to install

## npm

```bash:no-line-numbers
npm install --save-peer @aedart/support
```

## yarn

```bash:no-line-numbers
yarn add --peer @aedart/support
```

## pnpm

```bash:no-line-numbers
pnpm add --save-peer @aedart/support
```
