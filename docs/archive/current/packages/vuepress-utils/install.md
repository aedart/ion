# How to install

## npm

```bash:no-line-numbers
npm install --save-dev @aedart/vuepress-utils
```

## yarn

```bash:no-line-numbers
yarn add --dev @aedart/vuepress-utils
```

## pnpm

```bash:no-line-numbers
pnpm add --save-dev @aedart/vuepress-utils
```
